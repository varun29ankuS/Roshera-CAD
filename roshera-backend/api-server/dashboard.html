📚 API Documentation

Available Endpoints

POST /api/geometry/create

Create primitive geometry (box, sphere, cylinder, etc.)

Request Body:

{ "type": "box", "parameters": { "width": 10.0, "height": 10.0, "depth": 10.0 }, "position": [0, 0, 0], "material": "steel" }

Response:

{ "success": true, "geometry_id": "uuid", "properties": { "volume": 1000.0, "surface_area": 600.0, "bounding_box": {...} } }

POST /api/geometry/boolean

Perform boolean operations (union, intersection, difference)

Request Body:

{ "operation": "union", "operand_a": "geometry_id_1", "operand_b": "geometry_id_2" }

POST /api/ai/command

Process natural language CAD commands

Request Body:

{ "text": "create a box 10 by 20 by 30", "context": { "session_id": "optional", "previous_commands": [] } }

Response:

{ "success": true, "parsed_intent": "create_primitive", "parameters": {...}, "geometry_created": ["uuid"] }

POST /api/ai/vision

Process commands with viewport context

Request Body:

{ "text": "select the red box", "viewport": { "screenshot": "base64_image", "camera": {...}, "visible_objects": [...] } }

POST /api/sessions/create

Create a new design session

Request Body:

{ "name": "My Project", "description": "Optional description", "workflow": "part_maturity" }

WS /ws

WebSocket for real-time collaboration

GET /api/export/{geometry_id}/stl

Export geometry as STL for 3D printing

Query Parameters:

?format=binary&resolution=high
